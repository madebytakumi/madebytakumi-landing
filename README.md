## Presentación

`madebytakumi` es un estudio de producto digital enfocado en transformar ideas en soluciones claras, útiles y listas para operar. El enfoque combina ingeniería de software, criterio humano y flujos de trabajo potenciados por IA para avanzar con velocidad sin perder calidad, detalle ni mantenibilidad.

El sitio funciona como landing bilingüe y portafolio comercial para comunicar servicios, proyectos seleccionados, proceso de trabajo y canales de contacto.

## Soluciones

- **Presencia digital:** landing pages, catálogos digitales, blogs y contenido base para negocios que necesitan ser encontrados y generar confianza.
- **Venta online:** tiendas, catálogos con solicitud de pedido y experiencias responsivas para vender más allá del local físico.
- **Mantenimiento y automatización:** mejoras web, automatizaciones simples, integraciones y ajustes continuos para operar con más claridad.

## Trabajo Seleccionado

- **Exorno:** landing page premium para una empresa fabricante de cabinas de elevador.
- **Real-Time Booking API:** arquitectura backend para sistemas de reservación de alta demanda.
- **AI Workflow Demo:** automatización de procesos empresariales impulsados por IA.

## Método

1. **Discover:** entender el negocio, los clientes y el objetivo real del producto.
2. **Design:** convertir la idea en estructura, flujo, contenido y decisiones visuales.
3. **Build:** construir con ingeniería sólida y flujos potenciados por IA.
4. **Launch:** publicar, medir y ajustar para dejar una base lista para crecer.

## Stack

- **Framework:** Astro
- **Estilos:** Tailwind CSS + CSS variables
- **Animación:** Motion
- **Iconografía:** Lucide Astro
- **Salida:** sitio estático con sitemap

## Desarrollo

```bash
npm install
npm run dev
```

Comandos disponibles:

```bash
npm run dev      # Inicia el servidor local
npm run build    # Genera la versión de producción
npm run preview  # Previsualiza el build
```

## Estructura

```text
src/
  components/   Componentes reutilizables de interfaz
  data/         Datos de proyectos
  i18n/         Contenido bilingüe ES/EN
  layouts/      Layout base y metadatos
  pages/        Rutas del sitio
  sections/     Secciones principales de la landing
  styles/       Estilos globales y variables visuales
public/
  logo.svg
  og.svg
  favicon.svg
```
